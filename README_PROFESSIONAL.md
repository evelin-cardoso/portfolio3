# Portfolio Profissional — Evelin

Versão altamente polida do seu portfólio (estilo profissional).  
Arquivos prontos para deploy via GitHub Pages ou Netlify.

**O que incluir antes de publicar:**
1. Substitua `assets/profile.png` pela sua foto profissional (400x400 recomendado).
2. Substitua `assets/p1.png`, `p2.png`, `p3.png` pelas screenshots reais dos seus projetos.
3. Atualize textos e links nos arquivos HTML (procure por "ATUALIZAR AQUI").
4. Teste localmente abrindo `index.html`.

**Como publicar no GitHub Pages (resumido):**
- Crie repositório `meu-portfolio-profissional`.
- Faça commit dos arquivos e push para branch `main`.
- Em Settings → Pages, escolha branch `main` e / (root). Salve.
- Aguarde alguns minutos; seu site estará em `https://seu-usuario.github.io/meu-portfolio-profissional/`.

**Dicas de finalização:**
- Use fotos com boa iluminação, fundo neutro.
- Use screenshots de alta resolução (crops 1200x800).
- Escreva descrições concisas para cada projeto (problema, solução, tecnologias, resultado).
- Considere apontar domínio personalizado no futuro.

Feito com ❤️ — depois que revisar, posso:
- Gerar versões otimizadas das imagens;
- Subir para o GitHub usando orientações;
- Criar um vídeo de 1 minuto mostrando o site.
